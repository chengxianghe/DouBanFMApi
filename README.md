# DouBanFMApi
DouBan FM API 豆瓣FM的API

===
- 用 Objective-C 实现的豆瓣FM的简单播放器, 仅供学习交流使用. 
- 使用 ARC 和 CocoaPods 
- iOS 8.0
- 编译环境 Xcode 8.2

===

===

####API
[详细API](https://github.com/chengxianghe/DouBanFMApi/wiki)

现在项目里已经集成了 
频道列表,频道详情,我的红心,登录,取消/添加红心,歌词等接口.
